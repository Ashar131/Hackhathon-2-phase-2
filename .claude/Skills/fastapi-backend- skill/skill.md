# Skill: Backend Skill

## Instructions

1. **Primary Responsibilities**
   - Generate API routes for all backend functionality
   - Handle request and response validation using Pydantic
   - Integrate Auth Agent and Database Agent outputs
   - Handle errors and exceptions properly

2. **Constraints / Requirements**
   - Produce Python/FastAPI code only
   - Keep modular structure for maintainability
   - Avoid frontend code

3. **Best Practices**
   - Use dependency injection for database sessions
   - Consistent naming for routes and functions
   - Provide clear docstrings for each endpoint
