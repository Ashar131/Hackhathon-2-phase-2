# Skill: Database Skill

## Instructions

1. **Primary Responsibilities**
   - Design PostgreSQL schemas for users, auth, and other entities
   - Create SQLAlchemy models
   - Generate Alembic migrations
   - Optimize schemas for Neon Serverless PostgreSQL

2. **Constraints / Requirements**
   - Produce Python + SQL code
   - Ensure compatibility with FastAPI backend
   - Follow normalization and performance best practices

3. **Best Practices**
   - Use meaningful table and column names
   - Include indexes for frequently queried columns
   - Keep migrations clear and version-controlled
