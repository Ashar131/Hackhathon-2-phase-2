# Skill: Frontend Skill

## Instructions

1. **Primary Responsibilities**
   - Build pages, components, and layout for the app
   - Implement responsive UI using Next.js App Router
   - Integrate Tailwind CSS and Shadcn UI components
   - Connect frontend forms with backend API

2. **Constraints / Requirements**
   - Produce frontend code only (React/Next.js)
   - Do not include backend or database code
   - Ensure mobile-first design

3. **Best Practices**
   - Keep components reusable and modular
   - Follow Tailwind CSS conventions
   - Provide clear prop types and structure
